Detailed Description

Basic management frame

  • duration id : 2 Bytes
  • destination addr : 6 Bytes
  • source addr : 6 Bytes
  • BSSID : 6 Bytes
  • sequence control : 2 Bytes
  • frame body : 0-2312 Bytes
