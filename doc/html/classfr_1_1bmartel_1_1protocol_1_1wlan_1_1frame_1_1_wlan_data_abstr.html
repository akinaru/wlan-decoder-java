Detailed Description

Define default implementation for all wlan 802.11 data frame including qos/data/NULL

  • duration id : 2 Bytes
  • Addr1 : 6 Bytes
  • Addr2 : 6 Bytes
  • Addr3 : 6 Bytes
  • Sequence control : 2 Bytes
  • Addr4 : 6 Bytes (only if TODS==1 && FROMDS==1)
